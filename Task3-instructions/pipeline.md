# Инструкции для Agent по воспроизведению деталей Siemens NX через UGOPEN

## Цель

Данный документ описывает алгоритм, по которому agent должен уметь повторять построение любой новой детали в Siemens NX на основе чертежа, примера детали или отчёта.

Agent должен автоматически:

* анализировать структуру детали;
* выделять этапы построения;
* воспроизводить деталь через UGOPEN/Open C API;
* строить UI для запуска генерации модели;
* создавать компилируемый DLL-модуль.

---

# 1. Общий принцип построения деталей

Все детали строятся по одинаковому пайплайну:

```txt
Создание линий
↓
Создание эскиза
↓
Добавление геометрии в эскиз
↓
Вытягивание эскиза
↓
Создание отверстий через вычитание цилиндров
↓
Создание UI для запуска генерации
```

Agent обязан придерживаться именно этой последовательности.

# 5. Алгоритм анализа новой детали

При получении новой детали agent обязан:

## Шаг 1. Определить базовый профиль

Agent должен определить:

* внешний контур;
* количество линий;
* наличие дуг/окружностей;
* симметрию;
* базовую плоскость построения.

## Шаг 2. Выделить операции

Agent обязан определить:

### Основное тело

Создаётся через:

```txt
Эскиз + вытягивание
```

### Отверстия

Создаются через:

```txt
Вычитание цилиндров
```

### Дополнительные элементы

Если есть:

* выступы;
* карманы;
* ребра;
* фаски;
* дополнительные вытягивания;

они должны быть выделены как отдельные feature.

# 6. Стандарт построения детали

## Этап 1. Создание линий

Использовать:

```cpp
UF_CURVE_create_line
```

Agent должен:

* создавать точки;
* создавать линии;
* сохранять все tag_t.

### Шаблон

```cpp
UF_CURVE_line_t lineData;

lineData.start_point[0] = x1;
lineData.start_point[1] = y1;
lineData.start_point[2] = z1;

lineData.end_point[0] = x2;
lineData.end_point[1] = y2;
lineData.end_point[2] = z2;

UF_CURVE_create_line(&lineData, &lineTag);
```

---

## Этап 2. Создание эскиза

Использовать:

```cpp
UF_SKET_create_sketch
```

### Стандартная система координат

```cpp
double matrix[9] =
{
    1,0,0,
    0,1,0,
    0,0,0
};
```

### Добавление объектов

```cpp
UF_SKET_add_objects
```

---

## Этап 3. Вытягивание

Использовать:

```cpp
UF_MODL_create_extruded
```

## Стандартный шаблон

```cpp
uf_list_p_t objectList;
UF_MODL_create_list(&objectList);

UF_MODL_put_list_item(objectList, sketchTag);
```

### Направление вытягивания

По умолчанию:

```cpp
double direction[3] = {1.0, 0.0, 0.0};
```

### Симметричное вытягивание

Если в отчёте указано:

```txt
на 30 в обе стороны относительно 0
```

использовать:

```cpp
char start[] = "-30";
char end[] = "30";
```

# 7. Создание отверстий

## Правило

Любое отверстие создаётся:

```txt
цилиндр + UF_NEGATIVE
```

## Использовать

```cpp
UF_MODL_create_cyl1
```

## Булева операция

```cpp
UF_NEGATIVE
```

## Алгоритм

### Agent обязан определить:

* координату центра отверстия;
* ось отверстия;
* диаметр;
* глубину.

### Затем создать цилиндр:

```cpp
UF_MODL_create_cyl1(
    UF_NEGATIVE,
    origin,
    height,
    diameter,
    direction,
    &cylinderTag
);
```

# 10. Алгоритм воспроизведения новой детали

## Agent должен:

### 1. Выделить контур

Определить:

* линии;
* размеры;
* точки;
* симметрию.

### 2. Создать sketch

Добавить все линии в sketch.

### 3. Выполнить extrude

Определить:

* направление;
* длину;
* симметричность.

### 4. Создать отверстия

Для каждого отверстия:

* определить ось;
* определить координаты;
* создать цилиндр;
* выполнить вычитание.

### 5. Создать UI

Добавить:

* окно;
* кнопку;
* callback.

---

# 11. Правила генерации кода

## Agent обязан

### Генерировать:

* полностью компилируемый код;
* правильные include;
* корректные tag_t;
* освобождение памяти;
* раздельные функции.

## Запрещено

* псевдокод;
* пропуск UF_initialize;
* пропуск UF_terminate;
* memory leaks;
* смешивание UI и геометрии.

# 19. Критерии качества

Agent должен создавать код, который:

* компилируется;
* работает в Siemens NX;
* создаёт корректную геометрию;
* использует Open C API;
* не содержит утечек памяти;
* имеет модульную архитектуру;
* легко масштабируется под новые детали.
