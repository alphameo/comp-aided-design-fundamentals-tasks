# Запуск динамической библиотеки

1. Собрать `dll`
2. Создать пустую деталь
3. Из интерфейса NX: `Файл > Выполнить > NX Open... > [dll/path.dll]`
   Путь до `dll`

    ```
    x64/Release/Perekhodnik.dll
    ```

Файлы с разметкой интерфейса (берутся отсюда в runtime, не перемещать)

```
UI_NX/UI_Perekhodnik.(dlx|cpp|hpp)
UI_NX/preview.bmp
```
